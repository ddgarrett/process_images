```164:174:pi_filters.py
class FilterTbdBestPlusBest(Filter):
    ''' Filter rows for TBD Best candidates plus final Best '''

    def test(self,row:Row):
        if row['rvw_lvl'] == c.LVL_GOOD and row['img_status'] == c.STAT_TBD:
            return True

        if row['rvw_lvl'] == c.LVL_BEST and row['img_status'] == c.STAT_BEST:
            return True

        return False
```

You could use `LevelStatusFilter(c.LVL_BEST, c.STAT_BEST)` for `test`, but the base `Filter.get_descr()` returns `"xxx"`, and `PiFilterTable` uses `get_descr()` for the status line. So you either add a tiny subclass with a proper `get_descr()` (e.g. `"Best"`) or a small dedicated `FilterBest` class like the other show filters.
